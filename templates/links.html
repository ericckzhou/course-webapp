LaTeX

For LaTeX installations, go to the latex project install website. Both the Windows and Mac versions listed come with editors as well as the packages and compilers.

Once you are up and running, I recommend downloading the Not So Short Guide to LaTeX. Everything you need is there, however it may be overkill. You can also start with this template.tex and add as you need.
